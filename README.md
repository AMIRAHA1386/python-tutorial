## شروع برنامه نویسی
### نوشتن متن روی صفحه
```
print("hello world!")
```
### استفاده از متغیر
```
age = 20
print(age)
```
### گرفتن ورودی از کاربر
```
print("enter your age:")
age = input()
print("your age is:", age)
```
